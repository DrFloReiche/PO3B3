9.2.1 Conditional Probabilities

Emergence \[\begin{equation} P(y_{i,t} = 1 | y_{i, t-1} = 0) \end{equation}\]

Survival \[\begin{equation} P(y_{i,t} = 1 | y_{i, t-1} = 1) \end{equation}\]

9.2.2 Democratic Emergence

\[\begin{equation*} P(y_{i,t} = 1 | y_{i, t-1} = 0) \end{equation*}\]

  • The probability of a country to be a democracy in year \(t\), given that it was a dictatorship in the previous year, \(t-1\)

9.2.3

  • We therefore need a variable that gives us that information, the regime type in the previous year
  • We will use the global data set
  • Creating it, requires the package, as part of the tidyverse
library(haven)

world <- read_dta("files/Week 7/world.dta")

library(tidyverse)
  • We can now create the lagged democracy value
world <- world %>%
  group_by(countrykey) %>%
  mutate(l.democracy = lag(democracy)) %>%
  ungroup()
  • This creates a new variable in which all values are lagged by one year. As the first observation for each country cannot be lagged, it creates as many missing values as we have countries in the data set

Visualised:

9.2.4 What else?

  • The country needs to have been a dictatorship in the previous year
  • But we also need our independent variables
  • It is reasonable to assume, that the regime type in year \(t\) depends on the state of socio-economic development in the previous year, \(t-1\)
  • We therefore need to lag the values of all independent variables by one year

9.2.5 Lagging the Independent Variables

  • Same procedure as before.
  • For “per capita GDP” we call
world <- world %>%
  group_by(countrykey) %>%
  mutate(l.gdp_pc = lag(gdp_pc)) %>%
  ungroup()
  • Let’s do the same for: Life Expectancy:
world <- world %>%
  group_by(countrykey) %>%
  mutate(l.lifeexp = lag(lifeexp)) %>%
  ungroup()
  • and Primary Gross Enrolment Rate:
world <- world %>%
  group_by(countrykey) %>%
  mutate(l.enrl_gross = lag(enrl_gross)) %>%
  ungroup()

9.2.6 Subsetting the Data for Conditional Probabilities

  1. Select observations in which 
world_democ0 <- filter(world, l.democracy==0)
  1. Select observations in which 
world_democ1 <- filter(world, l.democracy==1)
  • Recall: \(P(y_{i,t} = 1 | y_{i, t-1} = 0)\)
  • Suppose, we want to see the effect of on democratic emergence.

9.2.7 Now we are ready!

emergence <- glm(democracy ~ l.gdp_pc, 
                 data = world_democ0, 
                 na.action = na.exclude,
                 family = binomial(link = "probit"))
